# JavaForStatement
for文

## 処理
1月から12月までを出力します。

## コード
```
public class For {

	public static void main(String[] args) {
		for (int i = 1; i <= 12; i++) {
			System.out.println(i + "月");
		}

	}

}
```

## 出力結果
```
1月  
2月  
3月  
4月  
5月  
6月  
7月  
8月  
9月  
10月  
11月  
12月
```
